Windows:
1) Abrir “Start.bat”
2) El navegador se abrirá en http://localhost:3000/admin.html

macOS:
1) Doble clic a “Start.command”
   - Si no abre: clic derecho → Abrir (primera vez por Gatekeeper)
2) El navegador se abrirá en http://localhost:3000/admin.html

Notas:
- Configurar SMTP en .env (copiar y renombrar .env.example si es necesario).
- Si en macOS pidiera permisos: ejecutar `chmod +x Start.command bin/node-macos-*`
